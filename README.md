# 團餐快手

這是一個團購點餐系統的第一版網頁原型，可以直接用瀏覽器開啟 `index.html` 使用。

## 目前功能

- 主揪建立團購單
- 上傳圖片菜單並顯示預覽
- 可按「辨識菜單」將圖片菜單轉成文字，再手動檢查後解析
- 上傳 TXT / CSV / MD / JSON 文字菜單到文字框，支援 UTF-8 / Big5 自動判斷
- 貼上文字菜單後可手動按「解析菜單」產生品項與價格
- 手動新增、修改、刪除菜單品項
- 產生同事點餐頁
- 同事填姓名、部門、餐點數量與備註
- 自動統計每人訂單、總金額
- 產生可複製給店家的訂單文字
- 資料暫存在瀏覽器本機

## 使用方式

1. 開啟 `index.html`
2. 在「主揪」頁上傳圖片菜單查看預覽
3. 可按「辨識菜單」轉成文字，檢查後按「解析菜單」
4. 確認品項後按「產生團購單」
5. 到「同事點餐」頁送出訂單
6. 到「統計」頁查看總表與店家訂單文字
